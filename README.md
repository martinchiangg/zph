zph
===

My dotfiles

Use at your own risk ;).

Installation
===
```shell
cd ~
git clone https://github.com/zph/zph.git ~/dotfiles
gem install homesick
homesick symlink ~/dotfiles
```

Pay attention to the prompts if you have conflicting files.
